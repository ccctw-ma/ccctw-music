# CCCTW Music Brand Assets

This folder keeps the current project logo system in SVG first, then exports platform-ready icon sizes from the same mark.

## Concept

- Warm cyan-blue circle keeps the icon brighter than the main dark UI.
- The single bold music note is the primary recognition element.
- No center play arrow is used, so the mark reads as music first rather than video.
- The transparent square canvas preserves a round app-icon silhouette across platforms.

## Assets

- Web favicon source: `apps/web/public/favicon.svg`
- Icon master: `apps/web/public/brand/ccctw-music-mark.svg`
- Horizontal lockup: `apps/web/public/brand/ccctw-music-logo-horizontal.svg`
- Intro/cover card: `apps/web/public/brand/ccctw-music-platform-intro.svg`
- Web manifest icons: `apps/web/public/brand/ccctw-music-icon-192.png`, `apps/web/public/brand/ccctw-music-icon-512.png`
- iOS touch icon: `apps/web/public/brand/ccctw-music-apple-touch-icon.png`
- Desktop icons: `apps/desktop/src-tauri/icons/`

## Usage

- Web: use `favicon.svg` and the manifest icons generated by Rsbuild `html.appIcon`.
- Mobile: use the 180, 192, and 512 PNG outputs as the Capacitor source icons before generating native app assets.
- Desktop: Tauri reads PNG, ICO, and ICNS files from `apps/desktop/src-tauri/icons/`.
- Project introduction: use `ccctw-music-logo-horizontal.svg` for compact headers and `ccctw-music-platform-intro.svg` for README, launch pages, or release notes.
